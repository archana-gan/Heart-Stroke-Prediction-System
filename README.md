# Heart-Stroke-Prediction-System
This Project is a machine learning–based system that analyzes patient health data (such as age, blood pressure, and medical history) to estimate the risk of stroke. It helps doctors and individuals take early preventive measures by identifying high-risk cases before a stroke occurs.

📊 Dataset: The dataset includes features like:
- Age
- Gender
- Hypertension
- Heart disease
- Average glucose level
- BMI
- Smoking status

🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

📈 Model: Common models used:

- KNN - k nearest neighbours
- Logistic Regression
- Decision Tree
- Random Forest
- Gaussian Naive Bayes'
- Bagging Classifier
- Support Vector Machine
- Stacking Classifer (KNN, Logistic Regression, Random Forest, Deviison Tree, Naive Bayes, Bagging Model, SVM)
- Multi-Layer perceptron (Neural Networks
- K-means Clustering
- Heirarchial Clustering
- Principal Component Analaysis (PCA)
- Singular Value Decompostiion
